Simple AI bot for Farcaster that replies when mentioned.

## Setup
1. Get API keys from [Neynar](https://neynar.com)
2. Deploy to Vercel
3. Set environment variables
4. Configure webhook in Neynar dashboard

## Environment Variables
- `NEYNAR_API_KEY`: Your Neynar API key
- `FARCASTER_SIGNER_UUID`: Your signer UUID

## Usage
Mention `@aibot` in any cast and get an AI response!
